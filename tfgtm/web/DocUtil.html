About

 

The application TheFirstGetTheMilk! serves as a shared shopping list from a Smartphone iOS, Android or Windows Phone. Compatible between different technologies, it enables you to easily add articles and notify other users of the list when adding or purchase an article. With the implementation TheFirstGetTheMilk !, users can better plan and manage their shopping list and save time.

 

1.       Creating a new shopping list

 

This feature allows you to create a shopping list from which you can add items to buy. The user enters a name for this list and click on the button "+".

From the "shopping lists" screen, positioned on the input field of a new list (the keyboard appears).

Enter a name to create the list and press "Done" (the keyboard disappears).

Press the "+" button and the list is created.

 

The list is then added below the input area.

 

2.       Deleting a shopping list

 

This feature allows you to delete a shopping list, whether empty or already contain articles.

From screen "shopping lists", drag left on the list to remove.

The "Delete" button appears to the right of the list.

Press the "Delete" button and shopping list disappear shopping lists.

 

3.       Adding an item to a shopping list by writing to the keyboard

 

This feature allows you to add an item to a shopping list.

Positioning on the page shopping lists.

Choose a shopping list by tapping it.

A screen shopping list is created, positioned on the input field of an article (keyboard appears).

Enter the tracking number of the article to add and press "Done" (the keyboard disappears).

Press the "+" button and the article is added below the input area accompanied by an icon representing the item.

When the item is added to a list, if the notification is enabled in the user preferences smartphone, a notification arrives on the smartphone.

 

 

4.       Adding an item to a shopping list by selecting it in the "Categories" screen

 

This feature allows you to add an item to a shopping list.

Press the button ">" to the right of "+" button.

From screen selecting by category, select a category in the drop down list of categories.

Under the list of categories, a list of items related to the selected category appears. Select an item to add to the shopping list by tapping it.

Press the button "<" and the selected item is added to the list of items in the shopping list.

 

When the item is added to a list, if the notification is enabled in the user preferences smartphone, a notification arrives on the smartphone.

 

 

5.       Removing an item from a list

 

This feature allows you to delete an item from a list in the case of a purchase or cancellation of adding an article.

From screen articles, slide to the left on the article to be deleted.

The "Delete" button appears to the right of the article.

Press the "Delete" button and section disappears from the shopping list.

 

 

6.       Share a shopping list with a user

 

After shopping list is created, this feature lets you add a user to the publication of his shopping list and to invite them to download the application, either by email or text message.

From the screen of the shopping list, press the "Share" button.

The TFGTM application opens the invitation view.

Enter an identifier.

Enter the email address of the person to invite or his mobile phone number. If "Invite by email" has been selected, the application opens TFGTM the default mail application installed on the device and opens an invitation email template to send a message to the user to invite. If a mobile number is entered, skip to step number 6, otherwise proceed to the next step.

Press "Submit" to confirm the invitation by email.

After indicating a cell phone number, press outside of the input fields to remove the keyboard.

Press "Invite by SMS" and TFGTM application sends a text message invitation to the selected user.

If you want to return to the application or send an invitation via SMS after sending an invitation by email, press "OK". If you have already sent the invitation text message, skip steps 9-11 inclusive.

Enter the mobile phone number of the user to be invited to send an invitation via SMS.

Press out the input fields to remove the keyboard.

Select "Invite by SMS" to accept the invitation. The TFGTM application sends sms to the desired user.
